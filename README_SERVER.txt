El archivo ChatServer.jar puede ubicarse en cualquier lugar de su computador.
Para ejecutarlo, abra una instancia de cmd.exe (Home + r, ingrese cmd.exe en
la caja de texto y presione enter). A continuación, ejecute el archivo .jar así:
java -jar "Dirección\de\instalación\ChatServer.jar". Ingrese el número del puer-
to que quiera utilizar (verifique que esté libre) y presione enter.
